# 广告数据自动化流程（对外版）

## 项目简介

这是一个广告数据自动化脚本集合，覆盖从数据拉取到展示分发的完整链路：  
**Adjust / Facebook 拉取 -> 数据合并 -> 飞书入库 -> 可视化展示 -> 群聊播报**。

## 核心能力

- 拉取 Adjust 与 Facebook 广告数据
- 自动合并多源报表并统一关键 ID
- 将结果写入飞书多维表格
- 提供本地与飞书数据看板
- 一键将图表发送到飞书群聊

## 目录（仅主流程 1~8）

```text
广告数据拉取/
├─ 1_Adjust拉取.py
├─ 2_FB拉取.py
├─ 3_表格合并.py
├─ 4_写入飞书表格.py
├─ 5_运行前四步main.py
├─ 6_图表展示_本地.py
├─ 7_图表展示_Lark.py
└─ 8_发送飞书群聊.py
```

## 流程说明

- `1`：拉取 Adjust 报表并做基础字段拆分
- `2`：拉取 Facebook 广告级指标
- `3`：按广告 ID + 日期合并两份数据
- `4`：分批写入飞书多维表格
- `5`：串行调度前四步（推荐入口）
- `6`：基于本地 CSV 启动可视化看板
- `7`：基于飞书数据启动可视化看板
- `8`：生成图表并发送飞书群聊

## 快速开始

1. 在运行环境中配置 API 凭证（Adjust / Facebook / 飞书）。
2. 进入 `广告数据拉取` 目录，执行：
   - `python 5_运行前四步main.py`
3. 如需看板：
   - 本地数据看板：`python 6_图表展示_本地.py`
   - 飞书数据看板：`python 7_图表展示_Lark.py`
4. 如需群播报：
   - `python 8_发送飞书群聊.py`

## 适用场景

- 广告投放日报自动化
- 多平台投放数据整合
- 团队群内数据可视化播报
